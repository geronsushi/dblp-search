Introduction¶

dblp-search is a simple search engine to query records for academic publications. The records are crawled from the famous academic search engine dblp.org by Scrapy.

Crawl and Indexing¶

According to the specifications of the project, three types of webpages from dblp.org are required to crawl. We can easily find that the start page for authors, journals and conference proceedings are well structured. With the help of xpath, the root webpage can be effectively analyzed where the links of related pages can be extracted. Following the links, we crawled all webpages by Scrapy.

The indexing module is designed following the design of Whoosh. For the crawled pages, we index the content from the 'publ-section' and the title. Given a specific query, the result is searched by matching the content between query and the indexed content by TF-IDF metric.

Note

We also provide the pagination scheme to support the multiple-page view of the query result. This is borrowed from Whoosh.

Webview¶

The webview module realized by Django. The web template is based on PythonDoc.
